# DebateX AI

A full-stack AI-powered debate platform built with **Spring Boot** and **React**.

## Tech Stack

- **Backend:** Java, Spring Boot, Maven
- **Frontend:** React, Vite
- **AI:** Google Gemini API

## Getting Started

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
